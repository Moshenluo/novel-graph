# 小说智能图谱 - 产品设计框架

## 📋 产品概述

**产品名称**：小说智能图谱 (Novel Graph Tool)  
**定位**：AI 驱动的小说世界构建与可视化管理工具  
**目标用户**：网络小说作家、剧本创作者、世界观设定爱好者  
**核心价值**：让复杂的人物关系和世界观设定可视化、结构化、智能化

---

## 🎯 产品愿景

### 使命
帮助创作者高效构建和管理小说世界，让灵感不再流失，让设定不再混乱。

### 愿景
成为每个创作者的"第二大脑"，让 AI 辅助创作成为标配。

---

## 👥 用户画像

### 核心用户
- **网络小说作家**（25-40 岁，男性为主）
  - 写作长篇连载小说（100 万字+）
  - 需要管理大量人物、势力、地点
  - 痛点：人物关系复杂易忘、设定前后矛盾

- **剧本创作者**（28-45 岁）
  - 创作影视/游戏剧本
  - 需要清晰的故事线和人物弧光
  - 痛点：多线叙事难以梳理、时间线混乱

- **世界观爱好者**（18-35 岁）
  - 喜欢构建原创世界设定
  - 享受设定过程本身
  - 痛点：设定分散在多处、难以系统性管理

## 🎨 设计原则

### 视觉设计
- **渐变色彩方案**：每种节点类型有专属配色
  - 人物：紫色 (#6366f1)
  - 组织：绿色 (#10b981)
  - 势力：红色 (#ef4444)
  - 地点：橙色 (#f59e0b)

- **统一设计语言**
  - 圆角：RADIUS.sm/md/lg/xl
  - 阴影：SHADOWS.md/lg
  - 间距：SPACING.xs/sm/md/lg/xl

- **响应式布局**
  - 左侧工具栏（可折叠）
  - 右侧详情面板（按需显示）
  - 中央画布/视图区域

### 交互设计
- **拖拽优先**：能拖拽就不点击
- **即时反馈**：所有操作有视觉反馈
- **撤销保障**：支持 50 步撤销/重做
- **智能提示**：关键操作有引导说明

## 🔄 用户流程

### 新手上路流程
```
1. 打开应用 → 空白画布
2. 从左侧拖拽"人物"卡片到画布
3. 输入人物名称 → 节点创建成功
4. 点击节点 → 右侧弹出详情面板
5. 填写人物简介、年龄、性别等
6. 拖拽另一个节点 → 建立第二个角色
7. 点击第一个节点的🔗按钮 → 拖拽到第二个节点
8. 选择关系类型（如"血缘"）→ 关系建立
9. 重复操作 → 构建完整人物关系网
```

### AI 辅助流程
```
1. 点击左侧"AI 灵感生成"
2. 输入描述："生成一个修仙世界的主角团"
3. 点击"AI 生成" → 等待 3-5 秒
4. AI 返回 3-5 个人物 → 自动添加到画布
5. 手动调整位置 → 建立关系
```

### 世界地图流程
```
1. 点击顶部"🗺️"进入地图视图
2. 输入世界描述："一个修仙世界，有五块大陆..."
3. 点击"AI 生成真实地形地图"
4. AI 生成不规则国境线 → SVG 渲染
5. 查看大陆列表 → 点击编辑详情
```

### 物品管理流程
```
1. 点击顶部"🎒"进入物品管理
2. 点击"➕ 添加物品"
3. 填写名称、类型、描述、威力
4. 选择拥有者（从人物列表）
5. 保存 → 物品出现在列表中
6. 在人物详情中查看"携带物品"
```

## 🛠️ 技术架构

### 前端技术栈
- **框架**：React 18 + TypeScript
- **构建工具**：Vite 5.x
- **图可视化**：@xyflow/react (ReactFlow)
- **状态管理**：React Hooks (useState/useReducer)
- **样式方案**：内联样式 + 设计系统常量
- **导出功能**：html2canvas

### AI 集成
- **API 提供商**：SiliconFlow
- **模型**：DeepSeek-R1
- **用途**：
  - 人物生成
  - 地点生成
  - 事件生成
  - 世界地图生成（国境线坐标）

### 数据存储
- **主要存储**：localStorage
- **数据格式**：JSON
- **数据结构**：
  ```typescript
  {
    nodes: Node[],
    edges: Edge[],
    characters: Character[],
    organizations: Organization[],
    forces: Force[],
    items: Item[],
    locations: Location[],
    worldRules: WorldRule[],
    events: TimelineEvent[],
    customRelations: CustomRelation[]
  }
  ```

### 性能优化
- **节点筛选**：useMemo 缓存搜索结果
- **历史记录**：限制 50 步防止内存溢出
- **懒加载**：视图按需渲染
- **防抖**：搜索输入防抖（待实现）
