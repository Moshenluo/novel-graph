# PRD：AI 小说转剧本 YAML 工具

## 目标

面向小说作者，提供一款 AI 辅助剧本创作工具，将 3 个章节以上的小说文本转换为结构化 YAML 剧本初稿，降低改编门槛，提升整理效率。

## 用户

- 小说作者：希望快速获得剧本初稿，但不熟悉剧本结构。
- 编剧协作者：需要从原文中提取场景、人物、对白和动作节拍。
- 评审与演示观看者：需要在短时间内复现核心功能。

## 核心流程

1. 选择改编目标：影视短片、电影长片、短剧分集、单集电视剧、舞台剧或有声剧。
2. 选择改编风格：保持原文、现实主义、悬疑惊悚、轻喜剧、古风权谋、情感爱情、科幻设定或奇幻史诗。
3. 导入小说文本：上传 `.txt`、`.md`、`.docx`，或直接粘贴正文。
4. 识别章节：自动拆分章节，并阻止少于 3 章的输入继续生成。
5. 生成草案：按章节生成场景卡片，提炼摘要、节拍、对白数量和人物候选。
6. 导出 YAML：输出符合 Schema 的剧本初稿，支持复制和下载。

## 非目标

- 不做完整在线协同编辑器。
- 不做复杂世界观规则系统。
- 不做旧版可拖拽关系图谱编辑器。
- 不承诺自动生成最终可拍摄剧本，只生成可编辑初稿。

## 验收标准

- 输入 3 个以上章节后，能完成草案查看与 YAML 导出。
- 未配置 API 时，仍能完成本地转换。
- 配置 API 时，人物和场景分析结果能进入 YAML。
- README、Schema、PR 汇总文档可支撑评审复现与过程检查。
