## 4. Demo 录制建议

录制时建议控制在 3 到 5 分钟：

1. 打开 README，说明题目要求和本项目主流程。
2. 启动项目，进入首页工作台。
3. 粘贴或导入至少 3 章小说文本。
4. 展示章节确认页，说明章节识别可人工复核。
5. 展示剧本草案页，说明人物共现图谱只是辅助检查，不抢主功能。
6. 进入 YAML 导出页，展示复制、下载和 Schema 说明。
7. 结尾说明项目不上传真实密钥，依赖和原创功能已在 README 列明。
