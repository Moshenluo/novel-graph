# 辅助功能备份记录

为突出竞赛主题“AI 小说转剧本工具”，当前主界面保留“文本导入 -> 章节确认 -> 剧本草案 -> YAML 导出”核心流程，并在剧本草案阶段保留人物共现线索作为辅助检查能力。

以下功能从主界面移除，仅作为后续扩展方向保留文字备份：

| 功能 | 原作用 | 后续恢复建议 |
| --- | --- | --- |
| 世界规则 | 管理修炼体系、地理、历史等设定。 | 可作为 `project.world_rules` 或独立 YAML 文件扩展。 |
| 物品管理 | 管理武器、道具、材料和线索。 | 可扩展为 `props` 字段，与场景引用关联。 |
| 时间线 | 统计和展示事件顺序。 | 可由 `scenes[].source_chapter` 和场景时间字段生成。 |
| AI 灵感与地图 | 调用外部接口生成创作素材。 | 当前已从主流程移除；若恢复，应通过后端代理和环境变量管理密钥。 |

当前版本不保留旧图谱时期的地图、世界规则、物品管理等外部创作接口。核心流程可在浏览器本地降级完成；若配置 `VITE_DEEPSEEK_API_KEY`，则启用 DeepSeek API 进行章节校对、人物识别、场景丰富和 YAML 片段精修。
