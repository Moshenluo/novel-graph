# PR 汇总：优化选区提示与面板间距

## 标题

优化 YAML 选区提示、导出区间距与人物图谱面板留白。

## 功能描述

本 PR 聚焦工作台 UI 可读性。YAML 导出区在作者选中片段后会显示更直观的选区状态、字符数和片段预览，帮助作者确认 AI 精修将作用于哪一段内容。

同时拉开 YAML 导出操作区、质量检查面板与人物共现图谱内部间距，缓解按钮、标题和卡片贴得过紧的问题。

## 实现思路

1. 将原本的文本式选区提示升级为状态条，区分未选中和已选中两种状态。
2. 已选中时展示字符数与压缩后的 YAML 片段预览，降低误操作概率。
3. 调整导出区双列 grid、操作按钮组和右侧检查面板间距。
4. 调整人物共现图谱标题、AI 校对标记与图谱卡片之间的留白。

## 测试方式

1. 运行 `npm run lint` 检查代码规范。
2. 运行 `npm run build` 检查 TypeScript 与生产构建。
3. 打开本地页面，确认 YAML 导出区左右面板不再贴紧。
4. 在 YAML 文本框中选中片段，确认选区状态条会显示字符数和预览。
