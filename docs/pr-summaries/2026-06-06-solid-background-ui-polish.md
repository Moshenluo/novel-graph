# PR 汇总：收敛单色背景与编辑区视觉噪音

## 标题

将工作台背景改为单色，并降低输入区视觉噪音。

## 功能描述

本 PR 根据界面反馈移除页面背景、上传区和小说输入框中的网格/纹理背景，让工作台更像安静的剧本编辑台。主流程仍保持“文本导入 -> 章节确认 -> 剧本草案 -> YAML 导出”，但视觉重心更集中在文本和 YAML 内容本身。

## 实现思路

1. 将 `body` 背景从网格渐变改为单色纸面背景。
2. 将主工作卡片改为不透明纸面白，减少叠层灰感。
3. 移除上传区域 hover 时出现的纹理覆盖层。
4. 将小说输入框背景改为纯色，保留 focus 边框和阴影反馈。
5. 轻微收敛主卡片、上传区和输入框边框/阴影强度。

## 测试方式

1. 运行 `npm run lint` 检查代码规范。
2. 运行 `npm run build` 检查 TypeScript 与生产构建。
3. 打开本地页面，确认背景为单色，输入区和上传区不再出现网格纹理。
4. 在移动端宽度下确认单列布局和输入框仍保持可读。
