# PR 汇总：Creative Production UI 工作台重构

## 标题

使用 Creative Production 风格方向重构 UI 为专业剧本工作台。

## 功能描述

本 PR 使用 Creative Production 插件梳理视觉方向，将界面从普通工具页升级为“AI 小说转剧本 YAML”的专业创作工作台。新的视觉方向强调主工作区优先、右侧检查面板、暖白纸面、墨色文本和少量胶片金，避免营销式 hero、旧图谱主导和配置表单占位。

## 实现思路

1. 使用 Creative Production 风格 intake 明确 UI 关键词：专业创作工作台、电影剧本编辑室、主工作区优先、右侧检查面板。
2. 新增 `studio-*` CSS 类，统一输入、草案、导出三类主网格。
3. 调整全局背景、header、侧栏和主卡片，使页面更像创作工作台而非模板化工具页。
4. 将主界面标题从 `novel-graph` 改为“小说转剧本 YAML”，强化竞赛主题。
5. 保留现有业务逻辑，不改动 YAML 生成、人物图谱和 AI 调用链路。

## 测试方式

1. 执行 `npm run lint`，确认 JSX 与样式引用无错误。
2. 执行 `npm run build`，确认生产构建通过。
3. 打开本地页面，确认四步流程仍可使用。
4. 检查桌面和窄屏布局，确认输入、草案、导出页主区域优先展示，辅助面板不抢主功能。
