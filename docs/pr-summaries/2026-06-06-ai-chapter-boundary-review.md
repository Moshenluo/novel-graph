# PR 汇总：新增 AI 章节边界校对

## 标题

新增 AI 章节校对，修正篇目式小说的章节划分。

## 功能描述

本 PR 优化“章节确认”步骤。原本章节划分主要依赖本地正则规则，面对《呐喊》这类短篇集或篇目式文本时，可能把目录、序言、篇名或正文切错。

现在作者可以在第二步点击“AI 校对章节”，让 AI 基于原文重新判断章节/篇目边界。校对成功后，系统会把文本重写为稳定的分章格式，后续人物图谱、剧本草案和 YAML 生成都会基于校对后的章节继续推进。

## 实现思路

1. 新增 `aiReviewChapterBoundaries`，把本地初切结果和原文片段交给 DeepSeek 校对。
2. AI 返回章节数组，字段包含 `title`、`content`、`reason` 和 `confidence`。
3. 新增 `formatReviewedChapters`，把 AI 校对结果写回统一的“第 N 章 标题 + 正文”格式。
4. 在章节确认页新增“AI 校对章节”按钮和说明提示。
5. 校对后清空旧 YAML、旧人物识别和生成统计，避免后续结果沿用旧章节边界。

## 测试方式

1. 运行 `npm run lint` 检查代码规范。
2. 运行 `npm run build` 检查 TypeScript 与生产构建。
3. 在章节确认页点击“AI 校对章节”，确认按钮进入 loading 状态。
4. 校对成功后确认章节数量、标题和正文预览更新。
5. 继续进入后续步骤，确认人物图谱和 YAML 生成基于新章节结果。
