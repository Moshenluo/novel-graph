# PR 汇总：补齐最终提交与 Demo 准备文档

## 标题

补齐 Demo 录制脚本与最终提交核对表。

## 功能描述

本 PR 做最终收尾文档整理。README 的 Demo 段落不再保留泛泛的 TODO，而是明确提示最终提交前替换为公开视频链接，并新增 Demo 录制脚本和最终提交核对表入口。

这能帮助最终提交时检查题目要求、演示覆盖点、PR 规范、依赖说明和密钥安全，降低遗漏风险。

## 实现思路

1. 新增 `docs/demo-script.md`，按 3-5 分钟视频结构列出演示顺序和建议话术。
2. 新增 `docs/final-readiness-checklist.md`，覆盖题目要求、功能演示、Demo 视频、开发过程、安全依赖和构建验证。
3. 更新 README Demo 区域，指向脚本和核对表。
4. 更新 PRD 中 Demo 准备状态，明确只剩最终公开视频链接替换。

## 测试方式

1. 运行 `npm run lint` 检查代码规范。
2. 运行 `npm run build` 检查生产构建。
3. 检查 README、Demo 脚本和最终核对表链接是否可读。
4. 运行敏感信息扫描，确认未提交真实 API Key。
